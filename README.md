# Daggerheart-Character-Sheet

This is a small project made to replace the paper format of the character sheet for Daggerheart.

This is NOT a full implementation, and I do not intend to fully automate the stats on the sheet (yet).



Warning: this project was coded using opencode. Yeah, I know, I don't like that I did this too, but I kinda had to, given I wanted to finish this as fast as possible.
