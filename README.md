# Daggerheart Character Sheet

This is a small personal project made to replace the paper format of the character sheet for Daggerheart.

This is NOT a full implementation, and I do not intend to fully automate the stats on the sheet (yet).



Warning: this project was coded using opencode. Yeah, I know, I don't like that I did this too, but I kinda had to, given I wanted to finish this as fast as possible.

As a result, I've kept the SPEC files, which don't have all the features, but most of them, as well as the "changelog" and "documentation" to be used in the future.


This project contains reference content from Daggerheart, by Darrington Press / Critical Role. 
More information on this in the NOTICE file
