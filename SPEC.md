# Daggerheart Character Sheet Application
Please create a python web application that contains a few fields that can be edited.
The edits for all fields shall be saved by the use of a button at the top-right of the page.
The fields can be either numeric (as a number) or text, and may have a checkmark next to them. Numbers can be negative.
The fields are separate, even when enumerated in the same line.
The fields shall be defined as follows, first being the displayed name of the field(s), and then the type of field:
- Name, Pronouns, Heritage, Subclass - text fields;
- Level - numeric field;
- Evasion - numeric field;
- Armor, Spent armor - numeric fields;
- Agility, Strength, Finesse, Instinct, Presence, Knowledge - numeric fields, with checkmark next to each of them;
- Damage Threshold 1, Damage Threshold 2 - numeric fields;
- HP, Marked HP, Stress, Marked Stress - numeric fields;
- Hope - numeric field;
- Experience 1, Experience 2, Experience 3, Experience 4, Experience 5 - text fields;
- Gold, Handfuls, Bags, Chests - numeric fields;
- Weapon Proficiency - numeric field;
- Primary Weapon - text field;
- Trait, Range, Damage Dice, Type - text fields;
- Name, Feature - text field;
- Secondary Weapon - text field;
- Trait, Range, Damage Dice, Type - text fields;
- Name, Feature - text field;
- Armor - text field;
- Base Threshold 1, Base Threshold 2, Base Score - numeric fields;
- Name, Feature - text fields;
